# User Practice API Playwright Tests

Boilerplate Playwright API tests for the User Practice API.

## Setup

Install dependencies:

```bash
npm install
```

## Environment

The project intentionally has no fallback defaults env vat. If either variable is missing, Playwright exits before running tests.

## What Is Included

- `tests/api/tester-todos.spec.js`: TODO placeholders where testers create more cases
